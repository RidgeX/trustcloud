[Commands]
Two newline terminated lines followed by binary data. The length of the binary data is specified on the first line as an ASCII unsigned integer. Example command:

8\n
PUT|test.txt|0\n
testdata

Add/replace file command:
<file length>\n
PUT|<filename>|0\n
<file>

Add/replace file response:
0\n
OK\n

Add/replace certificate command:
<file length>\n
PUT|<filename>|1\n
<file>

Add/replace certificate response:
0\n
OK\n

Fetch command:
0\n
GET|<filename>|<min-ring-length>\n

Fetch response:
<file length>\n
OK\n
<file>

List files command:
0\n
LIST\n

List files response:
<description length>\n
OK\n
<description>

Hash command:
0\n
HASH|<filename>\n

Hash response:
20\n
OK\n
<SHA1 hash bytes>

Vouch file command:
128\n
VOUCH|<filename>|<certname>\n
<signature>

Vouch file response:
0\n
OK\n

[Responses]
Same format as commands. Example response:
15\n
OK\n
Example message

Success response:
<binary data length>\n
OK\n
<binary data>

Fail response:
<binary data length>\n
FAIL\n
<binary data>
